# file-based-chat-logger

## Project Overview
This project is a simple Chat System built using Python.  
It allows users to send messages, view chat history, and clear chat data.
All messages are stored in a file (`chat.txt`) along with timestamps.

## Features
- Send messages  
- View chat history  
- Clear chat history  
- Timestamp for each message  
- Persistent storage using file handling  

## Technologies Used
- Python  
- datetime module  
- File Handling  

## How the Program Works
### 1. Send Message
- User enters a message  
- Message is stored with current time  
Example:[12:30:45] harsha: Hello

### 2. View Chat
- Reads all messages from file  
- Displays chat history  

### 3. Clear Chat
- User confirms action  
- File is cleared  

### 4. Exit
- Stops the program  

## How to Run
1. Install Python   
3. Run the program:chat looger.py
